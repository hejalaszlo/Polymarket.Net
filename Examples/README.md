# Examples

| Project | Topic |
|---|---|
| `Polymarket.Examples.Api` | Minimal ASP.NET Core API integration, dependency injection, public last-trade-price endpoint, private balance allowance endpoint |
| `Polymarket.Examples.Console` | Basic REST last-trade-price request and WebSocket token subscription |
| `Polymarket.Examples.OrderPlacement` | CLOB order placement, order status lookup, and cancellation |
| `Polymarket.Examples.OrderBook` | Client-side order book implementation with live console display |
| `Polymarket.Examples.Tracker` | Trade tracker with rolling market statistics |

The `ai-friendly` folder contains standalone copy/paste examples intended for quick onboarding and AI coding assistant context.
