# Examples

### Polymarket.Examples.Api
A minimal API showing how to integrate Polymarket.Net in a web API project

### Polymarket.Examples.Console
A simple console client demonstrating basic usage

### Polymarket.Examples.OrderBook
Example of using the client side order book implementation

### ai-friendly
Single-file examples optimized for AI assistants and quick onboarding. They cover market data, authentication and trading, WebSocket subscriptions, dependency injection, local order books, and error handling.
